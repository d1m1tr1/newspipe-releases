# newspipe-releases

Public release artifacts for [NewsPipe](https://github.com/d1m1tr1/newspipe-app)
(source repo is private). This repo holds:

- `appcast.xml` — Sparkle update feed consumed by the Mac app's `SUFeedURL`.
- GitHub Releases tagged `v<version>` with the signed `.app.zip` payloads.

To cut a release, see `RELEASE.md` in the source repo.
